# Proyecto Final de Backend con NodeJS/Express
Final Backend project for the NodeJS/Express course

---

[Consigna completa](./Consigna.md) (ver. español) 

Incluye:

- Arquitectura multicapa: rutas, controladores, servicios, modelos
- Middlewares: cors, body-parser
- Configuración de variables de ambiente con dotenv
- Contraseñas con bcryptjs
- Login con capa de seguridad con JWT (JSON Web Token)
- FireBase/FireStore
- Depliegue en Vercel
